Posso configurare quali test fare girare mediante i tag e poi tramite configurazione.
c.filter_run_including :focus => true
Oppure
c.filter_run :current_example
dipende se voglio essere più verboso o meno.

Analogamente posso anche escludere dei test che sono stati taggati in un modo particolare:
c.filter_run_excluding :broken => true

NB:
Posso usare i metadati :if e :unless senza bisogno di configurare niente.
:if => true il test gira
:if => false il test non gira
Se non metto niente è come mettere :if => true

Le keyword :if e :unless possono essere usate in concomitanza con altri filtri in modo da poter creare filtri composti.

Posso anche filtrare in maniera ancora più raffinata.
it 'run', :if => RUBY_VERSION == '2.1.1' do
    p RUBY_VERSION
end
