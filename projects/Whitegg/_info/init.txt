Installare Ruby nuovo:
* rvm install 2.3.1

Rimuovere le gemme dal gemset globale:
* cd  /home/xpuser/.rvm/gems/ruby-2.3.1@global
* rm -rf gems/*

Inizializzare il progetto
* cd /home/xpuser/mauro-quaglia/studio/projects
* rvm gemset create whitegg
* rvm gemset use whitegg
* gem install bundler --no-rdoc --no-ri
* gem install rails --no-rdoc --no-ri
* rails new Whitegg -T

Aggiungere al Gemfile le gemme:
* 'rspec-rails' al gruppo (:development, :test)
* 'therubyracer' per l'ambiente JavaScript
* 'bundler' con la versione fissata
* 'rails' con la versione fissata

Installare tutte le gemme:
* bundler

Inizializzare i test di RSpec:
* rails generate rspec:install

Aggiungere al progetto i file seguenti per fissare la versione di Ruby e il gemset:
* .ruby-version (ruby-2.3.1)
* .ruby-gemset  (whitegg)
